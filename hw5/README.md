# Usage 
1. C: clear all
2. +/-: invert input
3. %: percent(ie. *0.01)
4. +,-,x,/: operations
5. =: output
6. .: decimal
7. multiply/divide first and then substract/add 
   example "1" > "+" > "2" > "x" > "3" > "=" output: 7   
8. update screen when multiple operators clicked
9. handle exception: infinity/-infinity
